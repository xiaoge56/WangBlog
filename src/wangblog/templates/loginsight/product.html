崩溃信息日志

支持编程序语言SDK日志接入功能， 我们提供了异常信息抓取发送到服务器端功能， 服务器端会将这些异常信息或崩溃日志汇总，可以查看崩溃信息的堆栈， 可以设置告警规则（邮件告警）。

主流平台及框架支持

Node.js \ PHP \ Python \ Ruby \ Java \ Go \ Javascript \ C# \ Object-C

日志统一储存

日志数据长期保存，支持自研究的logagent日志接入，支持rsyslog/nxlog日志接入，支持日志仓库管理（流览、下载），支持实时日志数据推送。

实时日志

用户上传实时日志提供了采集日志的客户端程序（logagent), 用来监控应用服务器产生的日志， 将应用服务器日志实时发送到我们的存储队列中。同时, 我们也支持rsyslog/syslog nsyslog 实时日志监控发送的功能。
